InfoNES PPC2003 v0.97J

はじめに

　InfoNES PPC2003は、PPC2003で動作するファミリーコンピュータ(TM)のエミュレータです。

更新状況

v0.97J (05/09/03)

  • 操作性が向上しました
    • PPC2003:キー設定を修正（Pocket LOOX v70対応）

    仕様

    CPU

    • 6520オペコード
    • IRQ/NMI/フレームIRQ
    • SRAMロード/セーブ機能
    • トレーナ機能
    • 6502バグ
    • 6520未公開命令(一部)

    APU

    • 三角波
    • ノイズ
    • DPCM
